<strong>Version 26.05.11</strong>

[new] M-One standard has been tested and verified to work with this release.
[new] System setting tab now has a button to request System data.

[fixed] Editor would crash while setting different model from the User Interface.
[fixed] Program is sent to hardware when name is altered.

[improvement] Graphics flickering reduced.
[improvement] Editor initializes a lot faster.

[note] Tested and verified to work with with M-One Firmware 1.10 and M-One XL Firmware 2.09
[note] Global MIDI Channel should be set to 1.
[note] Device ID should be set to 0.
