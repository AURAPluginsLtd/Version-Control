<strong>Version 25.03.16</strong>

[new] Complete overhaul of the Pulse 2 Editor with Apple Silicon support.
[new] Windows AAX, VST2, VST3 and Standalone Application for Win 7 and greater.
[new] macOS AU, AAX, VST2, VST3 and Standalone Application for macOS 10.11 and greater.
[new] User Interface redesign with two skins: Black or Limited.
[new] Realtime resizable graphics.
[new] Editor and Librarian has dedicated 20-to-50 step Edit History.
[new] MIDI Learn per CC control can be defined with right clicking on parameter.
[new] Green LED is lit next to parameter name if MIDI CC is assigned.
[new] Randomizer dice to randomize each section with one click.
[new] Section controls to allow Section based presets and functions.
[new] Multi selection of items in librarian with shift+click or shift+arrow keys.
[new] Copy and Paste items with normal copy / paste commands.
[new] Onscreen Keyboard to trigger notes.
[new] MIDI mappings can be loaded or saved to hdd.
[new] Saving and Loading UI DATA which is essentially all parameters of the instance.
[new] Show tooltips on mouse hover.
[new] Show Bubble control of current parameter when adjusting value.
[new] Show parameter values on Editor display when mouse is over the control.
[new] Quick Edit view in Patch Librarian to adjust critical parameters while browsing the library.
[new] Improved Pulse 1 import function.

[fixed] All previous bugs with the Editor should be cleared now.

[other] User Manual is not yet available. Use tooltips to show function purpose. Most are self-explanatory. 
[other] Windows: Our signing certificate expired, so use right click -> open to install. This might affect to AAX compatibility on Windows as well. We will build new installers when we get a signing certificate.
