<strong>Version 26.01.11</strong>

[new] Multi select read and write options added to librarian bank edit menu and right click menu.
[new] MIDI I/O can be stored as defaults in PREFS file via Utilities menu.
[new] Randomizer settings are stored in the PREFS file upon closing Advanced MIDI Settings view.
[new] Automatic Update check can be enabled/disabled via Utilities menu.
[new] LOG status display now shows Editor status, ie. Ready, Hardware not detected etc.

[fixed] MIDI Channel change didn't route MIDI to hardware as expected.
[fixed] Resolved an issue where Editor would crash opening the UI while DAW playback was on.

[improvement] License system re-worked - no action needed from users.
[improvement] Drawing the UI elements re-worked a bit to reduce drawing issues on certain setups.
[improvement] Double click setting via Utilities menu is now working much better.

[other] Product URLs updated to support upcoming User Manual update. Use tooltips to show function purpose. Most are self-explanatory.
[other] Windows: Our signing certificate expired, so use right click -> open to install.
